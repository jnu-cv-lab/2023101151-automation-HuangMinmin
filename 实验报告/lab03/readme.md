# 实验三 图像下采样与恢复的频域分析实验

## 一、实验概述
本实验基于 OpenCV 实现图像的下采样与恢复，并从空间域、傅里叶频域、DCT 频域三个维度对不同处理方法的效果进行量化与可视化分析，完整覆盖图像插值、滤波、频域变换的核心知识点，严格遵循实验要求完成所有操作与分析。

## 二、实验环境
- 编程语言：Python 3.x
- 核心库：OpenCV（cv2）、NumPy（numpy）、OS（os）
- 开发工具：VS Code / PyCharm

## 三、实验原理
### 1. 下采样与预处理
- 直接下采样：不做预滤波，直接通过 `cv2.resize` 以最近邻插值将图像缩小为原尺寸的 1/2、1/4，保留原始高频细节，但存在轻微频谱混叠。
- 高斯平滑下采样：先通过 `cv2.GaussianBlur` 进行低通滤波（1/2 缩小使用 3×3 高斯核，1/4 缩小使用 5×5 高斯核），再进行下采样，有效抑制频谱混叠，但会丢失高频细节。

### 2. 图像恢复（插值方法）
- 最近邻插值（`INTER_NEAREST`）：直接复制邻域像素，计算速度快，但易产生块效应，人工高频分量多。
- 双线性插值（`INTER_LINEAR`）：基于 4 邻域像素做线性加权平均，平滑自然，有效抑制混叠，无明显伪影。
- 双三次插值（`INTER_CUBIC`）：基于 16 邻域像素做三次多项式拟合，细节保留能力强，但易产生振铃伪影，计算复杂度高。

### 3. 空间域质量评价
- MSE（均方误差）：衡量恢复图与原图的像素差异，值越低表示失真越小。
  $$MSE = \frac{1}{H \times W} \sum_{i=0}^{H-1} \sum_{j=0}^{W-1} (I(i,j) - K(i,j))^2$$
- PSNR（峰值信噪比）：基于 MSE 的客观评价指标，值越高表示图像质量越好。
  $$PSNR = 10 \log_{10} \left( \frac{MAX_I^2}{MSE} \right)$$

### 4. 傅里叶变换分析
- 对图像进行二维傅里叶变换，通过 `np.fft.fftshift` 将频谱中心移至图像中心，对幅度谱取对数以压缩动态范围，实现可视化。
- 频谱外围对应图像高频分量（边缘、纹理、细节），中心对应低频分量（整体亮度、平滑区域），用于分析不同处理对高频成分的影响。

### 5. DCT 变换分析
- 对图像进行二维 DCT 变换，DCT 系数图左上角为低频区域，右下角为高频区域。
- 统计左上角 1/16 区域的能量占总能量的比例，量化图像能量分布，低频占比越高表示高频丢失越严重。

## 四、实验步骤
1.  图像读入与预处理：读入彩色图像，转换为灰度图并保存，获取原始尺寸用于后续恢复对齐。
2.  下采样处理：生成 4 组缩小图（1/2 直接缩小、1/2 高斯缩小、1/4 直接缩小、1/4 高斯缩小）并保存。
3.  图像恢复：对 4 组缩小图分别使用最近邻、双线性、双三次插值恢复至原始尺寸，生成 12 组恢复图并保存。
4.  空间域比较：计算所有恢复图的 MSE 与 PSNR，控制台输出对比结果，同时显示原图、缩小图、恢复图。
5.  傅里叶变换分析：对原图、缩小图、双线性恢复图计算二维傅里叶变换，生成中心对齐、对数显示的频谱图并保存，对比高频成分差异。
6.  DCT 分析：对原图和所有恢复图做二维 DCT，生成对数显示的 DCT 系数图并保存，统计低频区域能量占比，对比不同恢复方法的能量分布差异。

## 六、实验结果与分析
### 1. 空间域结果
- 插值方法对比：直接缩小场景中，双线性插值 MSE 最低、PSNR 最高；高斯缩小场景中，双三次插值表现最优；最近邻插值因块效应，所有场景下 MSE 最高、PSNR 最低。
- 预处理方式对比：1/2 小比例下采样中，直接缩小恢复的 MSE 整体低于高斯缩小；1/4 大比例下采样中，高斯缩小的抗混叠优势凸显，MSE 显著低于直接缩小。
- 下采样比例对比：下采样比例越大，MSE 越高、PSNR 越低，图像失真越严重。

### 2. 傅里叶频谱分析
- 原图频谱：高频区域（外围）亮度最高，高低频分布完整，细节最丰富。
- 缩小图频谱：直接缩小图保留部分高频，存在轻微混叠；高斯缩小图高频被大幅抑制，频谱更干净。
- 恢复图频谱：双线性恢复图高频成分远低于原图，证明插值无法恢复丢失的原始高频；高斯恢复图高频几乎消失，细节丢失严重。
- 差异原因：下采样会直接丢失高频细节，高斯预处理提前滤除高频以抑制混叠，插值仅能平滑填充像素，无法重建丢失的高频。

### 3. DCT 能量分布分析
- 低频占比规律：高斯预处理后恢复的图像，低频占比显著高于直接缩小；双线性插值低频占比最高，双三次次之，最近邻最低；下采样比例越大，低频占比越高。
- 差异原因：高斯预处理滤除高频，使能量集中于低频；双线性插值平滑特性进一步集中低频能量；最近邻插值产生人工高频，降低低频占比；下采样比例越大，高频丢失越多，低频占比越高。

## 七、实验结论
1.  插值方法的适配性：直接缩小场景适配双线性插值，高斯缩小场景适配双三次插值，最近邻插值因块效应不适合高质量恢复。
2.  预处理的适用边界：高斯预处理仅在大比例下采样中具有抗混叠优势，小比例下采样中会因细节丢失降低图像质量。
3.  频域与空间域的一致性：MSE/PSNR 指标、傅里叶频谱、DCT 能量分布三者完全统一，高频保留越完整，MSE 越低、PSNR 越高、低频占比越低。
4.  插值的本质局限性：所有插值方法仅能平滑填充像素，无法恢复下采样丢失的原始高频细节，恢复图质量始终低于原图。
