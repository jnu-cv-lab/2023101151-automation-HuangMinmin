# Image Enhancement Experiment
本项目是计算机视觉课程实践实验，基于Python+OpenCV实现多种图像增强算法，完成低对比度、含噪图像的增强处理，包含完整的算法实现、结果可视化与定量评价，满足课程实验要求。

## 内容介绍

本实验围绕图像增强核心需求，实现了多种经典算法，重点解决低对比度、含噪图像的增强问题，通过主观视觉与客观指标双重验证算法效果，同时完成自定义算法实现，深入理解图像增强底层逻辑。

核心功能：

- 自定义实现：全局直方图均衡、均值滤波

- 调用实现：CLAHE、高斯滤波、中值滤波、拉普拉斯锐化

- 组合增强：滤波→均衡、均衡→滤波两种策略对比

- 结果可视化：自动生成「处理结果图+灰度直方图」

- 定量评价：计算PSNR、SSIM两项指标，客观评估增强效果

##  核心算法说明

| 算法类型 | 具体算法 | 实现方式 | 核心作用 |
|----------|----------|----------|----------|
| 对比度增强 | 全局直方图均衡 | 自定义实现 | 拉伸灰度分布，提升全局对比度 |
| 对比度增强 | CLAHE（clipLimit=2.0/4.0） | OpenCV调用 | 局部自适应增强，避免过增强与噪声放大 |
| 滤波去噪 | 均值滤波（3×3/5×5） | 自定义实现 | 平滑去噪，简单易实现（边缘易模糊） |
| 滤波去噪 | 高斯滤波（3×3/5×5） | OpenCV调用 | 平滑去噪，边缘保留效果优于均值滤波 |
| 滤波去噪 | 中值滤波（3×3/5×5） | OpenCV调用 | 抑制椒盐噪声，边缘保留最佳 |
| 细节增强 | 拉普拉斯锐化 | OpenCV调用 | 增强边缘细节，提升图像清晰度 |
| 组合增强 | 滤波→均衡 / 均衡→滤波 | 组合实现 | 对比处理顺序对含噪低对比度图像的影响 |

## 📊 实验结果说明

### 定量评价指标

- PSNR（峰值信噪比）：单位dB，数值越高，图像失真越小

- SSIM（结构相似性）：取值0~1，越接近1，图像结构保留越完整

### 核心实验结论

- CLAHE 综合效果优于全局直方图均衡，clipLimit=2.0为最优参数

- 中值滤波对椒盐噪声抑制效果最佳，高斯滤波兼顾去噪与边缘保留

- 滤波核大小：3×3优于5×5，核越大细节丢失越严重

- 组合增强最优策略：滤波→均衡（先去噪再增强，避免噪声放大）
