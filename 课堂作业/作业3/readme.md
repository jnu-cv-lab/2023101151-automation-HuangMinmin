# 作业3 
DFT与DCT的关系及其在图像处理中的边界条件解释
# 作业目标
### 通过公式分析、边界延拓理解和简单实验，对比离散傅里叶变换(DFT)与离散余弦变换(DCT)的关系，理解它们各自隐含的边界条件假设，并分析为什么在图像压缩和图像处理任务中DCT往往比 DFT 更常用。
## 内容
### 1. 分别写出一维DFT和一维DCT(建议采用DCT-II)的数学定义，并说明:
（1）两者的基函数分别是什么

（2）两者输出系数的主要形式有何不同

（3）DCT与DFT在”频域分解”思想上有什么共同点
### 2. DFT与DCT的关系，从”信号展开”的角度说明DCT与DFT的关系。
（1）DCT是否可以看作某种特殊条件下的傅里叶分析

（2）DCT与偶对称延拓有什么关系

（3）为什么DCT中只出现余弦项，而没有正弦项或显式虚部
### 3. 延拓方式对比，对同一个序列，分别构造:
（1）按DFT 隐含假设得到的周期延拓示意

（2）按DCT 隐含假设得到的偶对称延拓示意

要求画图或列出扩展后的序列，并说明两者在边界处的差异。
### 4. 频谱对比，分别计算该序列的 DFT和DCT系数,并回答:
（1）哪一种变换下，能量更集中在低频部分

（2）原因是什么

（3）这种能量集中性与边界条件之间有什么关系提示:可以用“前若干个系数所占能量比例”作为比较指标。
